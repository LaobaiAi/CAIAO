# CAIAO Visual - 操作指南

CAIAO Visual 是一个可视化工作流编辑器，用于构建、管理和运行 AI 数据处理流水线（Pipeline）。采用节点式编程理念，让你通过拖拽和连线的方式快速构建复杂的数据处理流程。

---

## 核心概念

### 节点（Nodes）
节点是流水线的基本构建块，每个节点代表一个独立的功能单元：

- **Server Node（服务器节点）**：执行具体的数据处理任务，可以配置工具（Tool）和服务（Server）
- **Input Server Node（输入节点）**：提供流水线的初始输入数据
- **Output Server Node（输出节点）**：收集和展示流水线的最终输出
- **Merge Server Node（合并节点）**：合并多个上游节点的数据流
- **Pipeline Node（管道节点）**：可配置工具和服务的通用节点
- **Note（便签）**：画布上的文本注释，用于标注和说明

### 连线（Edges / Wires）
连线定义了节点之间的数据流向。数据从上游节点的右侧输出端口流向下游节点的左侧输入端口。

- 拖拽节点上的**圆形句柄（Handle）**可以创建新连线
- **右键点击连线**可以隐藏/显示或删除连线
- **中键点击连线**可以快速删除
- 支持**盲线（Blind Wire）**模式，允许隐藏不需要显示的连接

### 组（Groups）
可以将多个相关节点组合在一起，用彩色边框标识：

- **选中多个节点后右键 → Group** 创建分组
- **拖拽组标题**可以整体移动组内所有节点
- **双击组标题**可以重命名组

## 鼠标操作

| 操作 | 方法 |
|------|------|
| **选择节点** | 左键点击节点 |
| **多选节点** | 按住 Shift + 左键点击，或拖拽框选 |
| **移动节点** | 左键拖拽节点 |
| **移动视图** | 中键拖拽画布，或按住空格键 + 左键拖拽 |
| **缩放视图** | 滚轮缩放 |
| **创建连线** | 从节点句柄拖拽到另一个节点的句柄 |
| **框选** | 在画布空白处按住左键拖拽（部分选择模式） |
| **右键点击节点** | 打开节点上下文菜单 |
| **右键点击画布** | 打开画布上下文菜单 |
| **双击画布** | 关闭所有菜单 |
| **双击节点** | 打开节点属性面板 |

## 节点右键菜单

右键点击节点可以访问以下功能：

### 编辑（Edit）
- **Duplicate**：复制节点
- **Rename**：重命名节点
- **Nickname**：给节点设置别名
- **Set Color**：自定义节点边框颜色
- **Copy Config**：复制节点配置到剪贴板

### 分组（Group）
- **Group**：将选中节点组合为一个分组
- **Ungroup**：取消分组
- **Select All in Group**：选中同一组内的所有节点

### 对齐（Align）- 仅多选时
- Align Left / Center / Right（水平对齐）
- Align Top / Middle / Bottom（垂直对齐）

### 视图（View）
- **Properties**：打开节点属性面板（相当于双击）
- **Find References**：高亮所有与该节点相连的上下游节点
- **Zoom to Fit**：自适应视图
- **Export Result**：导出节点执行结果（运行后可用）

### 控制（Control）
- **Disable/Enable**：禁用/启用节点
- **Lock/Unlock**：锁定节点位置
- **Show/Hide Profiler**：显示/隐藏执行时间统计
- **Set/Remove Breakpoint**：设置/移除断点

## 画布右键菜单

右键点击画布空白区域可以访问：

- **Select All / Invert / Deselect**：全局选择操作
- **Group Selection**：将当前选中节点分组
- **Add Note**：在点击位置添加便签注释
- **Toggle Snap to Grid**：切换网格对齐模式
- **Show All Wires**：显示所有隐藏的连线

---

## 工作流（Flows）

### 创建和管理
- 左侧边栏 **CAIAO FLOWS** 面板列出所有工作流
- 点击 **+** 按钮创建新工作流
- 点击工作流名称在标签页中打开
- 工作流名称旁边的 **\*** 表示有未保存的更改
- 点击 💾 按钮保存工作流

### 自动保存
- 每次修改节点位置或连线后，系统会自动保存（延迟 1 秒）

---

## 组件面板（右侧边栏）

右侧组件面板列出了所有可用的服务器组件，按类别分组：

- **搜索**：在顶部搜索框中输入关键字筛选组件
- **折叠/展开**：点击类别标题折叠或展开
- **添加组件**：将鼠标悬停在组件上，点击 **+** 按钮添加到画布
- **切换分类**：将组件移到其他类别中

---

## 高级技巧

1. **高效布线**：先在左侧添加组件，再拖拽连线，从上游到下游逐步构建流水线
2. **使用分组**：将逻辑相关的节点分组，便于整体移动和管理
3. **便签注释**：在关键位置添加便签，记录说明文字
4. **书签导航**：在复杂工作流中使用视图书签快速跳转
5. **锁定节点**：锁定已完成配置的节点，防止误操作
6. **显示所有连线**：当需要查看完整的连接关系时使用 `Ctrl+M`

---

## 数据流水线运行

在流水线中运行节点处理数据：

1. 配置好所有节点的输入参数
2. 数据从 **Input Server** 流向各个处理节点
3. 中间节点处理完成后将结果传递给下游节点
4. 最终结果汇总到 **Output Server**
5. 查看节点的 **View Output** 获取详细结果

---

## 状态指示

每个节点右上角的状态指示器显示当前运行状态：

- ● **灰色** — 空闲（Idle）
- ● **蓝色闪烁** — 运行中（In Progress）
- ● **绿色** — 完成（Complete）
- ● **红色** — 错误（Error）

---

## 视觉提示

- **橙色边框**：节点已禁用（Disabled）
- **蓝色锁图标**：节点已锁定（Locked）
- **红色 [bp] 标记**：节点设置了断点（Breakpoint）
- **自定义颜色**：通过右键菜单为节点设置独特的边框颜色
- **组颜色**：不同分组自动分配不同颜色便于区识别
